# 🚌 Bus Ticket Booking System

A full-stack **Bus Ticket Booking System** developed using **Spring Boot**, **Thymeleaf**, and **MySQL**. The application enables users to search available buses, book tickets, manage reservations, and provides an intuitive interface for seamless travel booking.

---

## 🚀 Features

- 🔐 User Authentication & Authorization
- 🚌 Search Available Buses
- 🎫 Bus Ticket Booking
- ❌ Booking Cancellation
- 📋 Booking History
- 👤 User Profile Management
- ⚙️ Admin Dashboard
- 🪑 Seat Availability Management
- 📱 Responsive User Interface

## ⚙️ Getting Started

### Clone the Repository

```bash
git clone https://github.com/techayush087/bus-ticket-booking-system.git
```

### Navigate to the Project

```bash
cd bus-ticket-booking-system
```

### Configure MySQL

Update your database credentials in:

```text
Backend/src/main/resources/application.properties
```

### Build the Project

```bash
cd Backend
mvn clean install
```

### Run the Application

```bash
mvn spring-boot:run
```

The application will be available at:

```
http://localhost:8080
```
